# Digital KPI Intelligence Hub

Website tĩnh đọc file Excel ngay trên trình duyệt và tự động tạo:

- Dashboard KPI và ngân sách theo thương hiệu
- Biểu đồ cơ cấu chi tiêu theo kênh
- Brand Awareness đa nền tảng
- Xu hướng TikTok và Shopee
- Đánh giá KPI theo quy tắc
- Pivot Table tương tác
- Trình duyệt toàn bộ sheet
- Báo cáo lỗi công thức và chất lượng dữ liệu

## Cách đưa lên GitHub Pages

1. Tạo repository mới, ví dụ `digital-kpi-intelligence-hub`.
2. Upload `index.html` và `README.md` vào thư mục gốc.
3. Vào **Settings → Pages**.
4. Chọn **Deploy from a branch → main → /(root)**.
5. Website sẽ có dạng: `https://stevekhanh2404.github.io/digital-kpi-intelligence-hub/`.

## Quyền riêng tư

File Excel được đọc và xử lý cục bộ trong trình duyệt. Website không gửi file lên server.

## Thư viện

Website tải SheetJS và Chart.js từ jsDelivr CDN. Khi dùng lần đầu cần có kết nối Internet.
